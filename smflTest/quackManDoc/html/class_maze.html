◆ loadLevel()

void Maze::loadLevel ( string  a_fileName)

This function reads a 28 x 31 image(PNG) which represent the map, then read the pixel data and translate into a vector of sf::Vector2i. Each color in the image represent certain maze entity. Black are Walls, White are the pellets, Green are the super pellets, Red are the ghost spawn point, Blue is Pac-Man/QuackMan's spawn point, and Magenta is the ghost respawn point. This function also takes the maze data and renders wall sprites according to its position (Outer Corner, Inner Corner, Border).

Parameters
a_fileNamethe map image file name
